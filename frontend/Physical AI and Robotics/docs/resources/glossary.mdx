# Glossary

<!-- Add theory, examples, diagrams here -->

## A

**Action**: In ROS 2, a communication pattern for long-running tasks with feedback.

## B

**Behavior Tree**: A hierarchical structure for task planning and decision-making.

## D

**Digital Twin**: A virtual representation of a physical robot used for simulation and testing.

**DDS**: Data Distribution Service, the middleware used by ROS 2.

## G

**Gazebo**: An open-source robot simulation environment.

## I

**IMU**: Inertial Measurement Unit, a sensor measuring acceleration and angular velocity.

## L

**LiDAR**: Light Detection and Ranging, a sensor using laser pulses to measure distances.

## P

**Physical AI**: Artificial intelligence systems that interact with the physical world through embodied agents.

## R

**ROS 2**: Robot Operating System 2, a framework for robot software development.

## U

**URDF**: Unified Robot Description Format, an XML format for describing robot models.

## V

**VLA**: Vision-Language-Action model for end-to-end robotic control.

**VLM**: Vision-Language Model for understanding visual and textual information.
