## جائزہ

This section outlines the hardware requirements for following the AI-Native Robotics textbook.

## Minimum Requirements

- **RAM**: 8GB
- **CPU**: 4-core processor (Intel i5 or AMD equivalent)
- **Storage**: 50GB free space
- **Graphics**: Integrated graphics (Intel HD, AMD Radeon)
- **OS**: Ubuntu 22.04, Windows 10/11 (WSL2), macOS (Docker)

## Recommended Requirements

- **RAM**: 16GB or more
- **CPU**: 8-core processor (Intel i7/i9 or AMD Ryzen)
- **Storage**: 100GB SSD
- **Graphics**: Dedicated GPU (NVIDIA preferred for Isaac Sim)
- **OS**: Ubuntu 22.04 LTS (native installation)

## GPU Requirements for Isaac Sim

- NVIDIA GPU with RTX support
- CUDA 11.8 or later
- 8GB+ VRAM

## Network Requirements

- Stable internet connection for downloading packages and resources
