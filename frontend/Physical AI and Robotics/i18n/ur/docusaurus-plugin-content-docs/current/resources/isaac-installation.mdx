# NVIDIA Isaac Sim Installation

Complete guide for installing NVIDIA Isaac Sim for GPU-accelerated robotics simulation and AI training.

## جائزہ

NVIDIA Isaac Sim is a scalable robotics simulation platform powered by Omniverse, featuring:
- Photorealistic rendering with RTX ray tracing
- Accurate physics simulation
- ROS 2 integration
- Synthetic data generation for AI training
- Digital twin capabilities

## System Requirements

### Minimum Requirements

- **GPU**: NVIDIA RTX 2060 or higher
- **VRAM**: 8GB minimum
- **CPU**: Intel Core i7 or AMD Ryzen 7
- **RAM**: 32GB
- **Storage**: 50GB SSD
- **OS**: Ubuntu 22.04 LTS or Windows 10/11
- **Drivers**: NVIDIA Driver 525.60.11 or newer

### Recommended Requirements

- **GPU**: NVIDIA RTX 3080/4080 or A5000/A6000
- **VRAM**: 16GB or more
- **CPU**: Intel Core i9 or AMD Ryzen 9
- **RAM**: 64GB
- **Storage**: 100GB NVMe SSD
- **Drivers**: Latest NVIDIA Studio or Gaming drivers

## Installation Methods

### Method 1: Omniverse Launcher (Recommended)

#### Step 1: Install NVIDIA Drivers

**Ubuntu:**
```bash
# Add NVIDIA PPA
sudo add-apt-repository ppa:graphics-drivers/ppa
sudo apt update

# Install latest NVIDIA driver
sudo apt install -y nvidia-driver-535

# Reboot system
sudo reboot

# Verify installation
nvidia-smi
```

**Windows:**
- Download from [NVIDIA Driver Downloads](https://www.nvidia.com/download/index.aspx)
- Install and reboot

#### Step 2: Download Omniverse Launcher

1. Visit [NVIDIA Omniverse](https://www.nvidia.com/en-us/omniverse/)
2. Click "Download Omniverse"
3. Sign in with NVIDIA account (free)
4. Download installer for your OS

**Ubuntu:**
```bash
# Make installer executable
chmod +x omniverse-launcher-linux.AppImage

# Run installer
./omniverse-launcher-linux.AppImage
```

**Windows:**
- Run `omniverse-launcher-win.exe`
- Follow installation wizard

#### Step 3: Install Isaac Sim via Launcher

1. Open Omniverse Launcher
2. Navigate to "Exchange" tab
3. Search for "Isaac Sim"
4. Click "Install" (latest version: 2023.1.1)
5. Wait for download (~30GB)

#### Step 4: Install Dependencies

**Ubuntu:**
```bash
# Install required libraries
sudo apt install -y \
    libatomic1 \
    libegl1 \
    libglu1-mesa \
    libgomp1 \
    libxi6 \
    libxrandr2 \
    libxxf86vm1

# Install Python dependencies
pip3 install numpy==1.23.5
```

### Method 2: Docker Container

:::info
Docker installation is ideal for headless servers or CI/CD pipelines.
:::

#### پیشگی ضروریات

```bash
# Install NVIDIA Container Toolkit
distribution=$(. /etc/os-release;echo $ID$VERSION_ID)
curl -s -L https://nvidia.github.io/nvidia-docker/gpgkey | sudo apt-key add -
curl -s -L https://nvidia.github.io/nvidia-docker/$distribution/nvidia-docker.list | \
  sudo tee /etc/apt/sources.list.d/nvidia-docker.list

sudo apt update
sudo apt install -y nvidia-container-toolkit
sudo systemctl restart docker
```

#### Pull Isaac Sim Container

```bash
# Login to NVIDIA NGC
docker login nvcr.io
# Username: $oauthtoken
# Password: <Your NGC API Key>

# Pull Isaac Sim container
docker pull nvcr.io/nvidia/isaac-sim:2023.1.1

# Run container with display
docker run --name isaac-sim --entrypoint bash -it --gpus all \
  -e "ACCEPT_EULA=Y" \
  --rm --network=host \
  -e "PRIVACY_CONSENT=Y" \
  -v ~/docker/isaac-sim/cache/kit:/isaac-sim/kit/cache:rw \
  -v ~/docker/isaac-sim/cache/ov:/root/.cache/ov:rw \
  -v ~/docker/isaac-sim/cache/pip:/root/.cache/pip:rw \
  -v ~/docker/isaac-sim/cache/glcache:/root/.cache/nvidia/GLCache:rw \
  -v ~/docker/isaac-sim/cache/computecache:/root/.nv/ComputeCache:rw \
  -v ~/docker/isaac-sim/logs:/root/.nvidia-omniverse/logs:rw \
  -v ~/docker/isaac-sim/data:/root/.local/share/ov/data:rw \
  -v ~/docker/isaac-sim/documents:/root/Documents:rw \
  nvcr.io/nvidia/isaac-sim:2023.1.1
```

### Method 3: pip Installation (Python Package)

# Create virtual environment

# Install Isaac Sim (Note: Requires Omniverse Nucleus)

## ROS 2 Bridge Setup

### Install ROS 2 Packages

```bash
# Install Isaac ROS packages
sudo apt install -y \
  ros-humble-isaac-ros-common \
  ros-humble-isaac-ros-visual-slam \
  ros-humble-isaac-ros-image-pipeline

# Install additional ROS 2 dependencies
sudo apt install -y \
  ros-humble-navigation2 \
  ros-humble-nav2-bringup \
  ros-humble-ros2-control \
  ros-humble-ros2-controllers
```

### Configure ROS 2 Bridge in Isaac Sim

1. Open Isaac Sim
2. Go to **Window → Extensions**
3. Search for "ROS2 Bridge"
4. Enable the extension

**Enable via Python:**

```python
from omni.isaac.kit import SimulationApp

simulation_app = SimulationApp({"headless": False})

# Enable ROS2 bridge
from omni.isaac.core.utils.extensions import enable_extension
enable_extension("omni.isaac.ros2_bridge")

simulation_app.close()
```

### Test ROS 2 Connection

```bash
# Terminal 1: Start Isaac Sim with ROS 2 example
cd ~/.local/share/ov/pkg/isaac_sim-2023.1.1
./python.sh standalone_examples/api/omni.isaac.ros2_bridge/carter_stereo.py

# Terminal 2: Check ROS 2 topics
source /opt/ros/humble/setup.bash
ros2 topic list

# You should see Isaac Sim topics
ros2 topic echo /clock
```

## Troubleshooting

### Issue: "No GPU detected"

**Solution**: Verify NVIDIA driver installation
```bash
nvidia-smi
# Should show your GPU details
```

### Issue: Isaac Sim crashes on startup

**Solution**: Update graphics drivers and disable Vulkan
```bash
# Set environment variable
export ISAAC_SIM_FORCE_OPENGL=1
```

### Issue: ROS 2 bridge not working

**Solution**: Check ROS 2 environment
```bash
# Ensure ROS 2 is sourced
source /opt/ros/humble/setup.bash

# Verify ROS_DOMAIN_ID matches
echo $ROS_DOMAIN_ID
```

### Issue: Out of memory errors

**Solution**: Reduce scene complexity or batch size
```python
# Reduce physics update rate
world.reset()
world.set_simulation_dt(physics_dt=1.0/60.0)  # 60 Hz instead of 120 Hz
```

### Issue: Slow rendering performance

**Solution**: Disable ray tracing
```bash
# Launch with rasterization
~/.local/share/ov/pkg/isaac_sim-2023.1.1/isaac-sim.sh --/renderer/enabled=rtx --/rtx/enabled=false
```
