## Using `_category_.json` Files

To customize how folders appear in the sidebar, add a `_category_.json` file:

```json
{
  "label": "Chapter 1: Physical AI Fundamentals",
  "position": 1,
  "collapsed": false,
  "link": {
    "type": "generated-index",
    "description": "Introduction to Physical AI concepts."
  }
}
```

### Properties Explained

- **`label`**: Display name in the sidebar
- **`position`**: Numeric order (1, 2, 3... 999 for Resources)
- **`collapsed`**: Whether the category starts collapsed (true/false)
- **`link.type`**: `"generated-index"` creates an auto-generated index page
- **`link.description`**: Description shown on the index page

## Customizing Sidebar Order

To change the order of chapters, edit the `position` value in each chapter's `_category_.json`:

- Chapter 1: `position: 1`
- Chapter 2: `position: 2`
- Chapter 3: `position: 3`
- Chapter 4: `position: 4`
- Chapter 5: `position: 5`
- Resources: `position: 999` (to appear at the end)

## Troubleshooting

### If chapters appear out of order:
- Check `position` values in each `_category_.json`
- Ensure all chapters have `_category_.json` files

### If chapter labels look wrong:
- Update the `label` property in `_category_.json`

### If you need to exclude files:
- Prefix with underscore: `_internal-notes.md` (hidden from sidebar)
- Or add to `.docusaurus` ignore patterns
